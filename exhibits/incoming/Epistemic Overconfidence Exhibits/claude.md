the files in this folder are destined for exhibit E006_epistemic_overconfidence

This is a demonstration of epistemic overconfidence, using local gpt-oss-20b on LM-Studio.

Findings: The low-reasoning run demonstrates an incomplete reasoning process, resulting in a false belief that the 
model is capable of reasoning. The high-reasoning run demonstrates a complete reasoning process, resulting in a 
false statement that cheese would turn directly from a solid to a gas in the vacuum of space.

The medium and high-reasoning runs are both largely correct, but the high-reasoning run is more complete.
